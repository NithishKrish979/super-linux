# 🚆 Railway Ticket Reservation System

A modern Railway Ticket Reservation System developed using Python, CustomTkinter, and MySQL.  
This application allows users to book, search, update, view, and cancel railway tickets through a dark-themed graphical user interface.

---

## 📌 Features

✅ Add New Ticket  
✅ View All Tickets  
✅ Search Ticket by ID  
✅ Update Ticket Details  
✅ Cancel Ticket  
✅ Modern Dark-Themed GUI  
✅ MySQL Database Connectivity  

---

## 🛠 Technologies Used

- Python
- CustomTkinter
- MySQL
- MySQL Connector
